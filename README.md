# Scholarship
This is an iPad app that I use as a resume to apply for scholarships.

## Compilation
1. Install [Carthage](https://github.com/Carthage/Carthage). 
2. Run `carthage bootstrap`
3. Open `Scholarship.xcodeproj` and run the app.
